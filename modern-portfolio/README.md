# Modern Portfolio Website

This is a modern personal portfolio website built using React and Tailwind CSS. The website showcases various sections including Home, About, Projects, Publications, and Contact, designed to present information in a minimalist and professional layout.

## Features

- **Home**: A brief introduction with a professional photo placeholder and a dynamic typewriter effect for the tagline.
- **About**: A short bio detailing background, interests, and expertise in AI research, LLMs, and academic writing.
- **Projects**: A grid layout displaying key projects with titles, descriptions, and links to GitHub or publications.
- **Publications**: A section listing selected research works with titles, abstracts, and "Read more" links.
- **Contact**: A simple contact form for inquiries and links to social profiles (GitHub, LinkedIn, Google Scholar).
- **Light/Dark Mode**: A toggle feature for switching between light and dark themes.
- **Responsive Design**: Optimized for both desktop and mobile views.
- **Smooth Scroll and Animations**: Enhanced user experience with smooth scrolling and fade-in animations.

## Installation

To get started with this project, clone the repository and install the dependencies:

```bash
git clone <repository-url>
cd modern-portfolio
npm install
```

## Usage

To run the development server, use the following command:

```bash
npm start
```

The application will be available at `http://localhost:3000`.
